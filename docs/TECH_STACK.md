# Технологический стек Clan Portal

## Архитектурный подход

Для версии 1.0 рекомендуется модульный монолит: одно приложение, одна база данных и чётко разделённые предметные модули. Это быстрее и дешевле микросервисов, но не мешает позднее вынести импорт данных или уведомления в отдельные процессы.

## Основной стек

| Часть | Технология | Назначение |
| --- | --- | --- |
| Веб-фреймворк | Next.js 16 (App Router) | React-интерфейс, серверная логика, маршруты и API в одном проекте |
| Интерфейс | React 19 + TypeScript | Надёжные компоненты и контроль типов |
| Дизайн-система | CSS tokens сейчас; Tailwind CSS + shadcn/ui при развитии компонентов | Быстрая разработка без потери фирменного стиля |
| База данных | PostgreSQL | Пользователи, коллективы, активы, заявки, рецепты и история операций |
| Работа с БД | Drizzle ORM + SQL migrations | Прозрачная схема, типизация и контролируемые изменения базы |
| Авторизация | Better Auth + Discord OAuth | Вход через Discord и управление сессиями |
| Валидация | Zod | Одинаковые правила проверки данных на клиенте и сервере |
| Импорт предметов | Node.js worker + планировщик | Обновление предметов, рецептов и изображений без нагрузки на портал |
| Тестирование | Vitest, React Testing Library, Playwright | Проверка логики, компонентов и основных сценариев пользователя |
| Развёртывание | Docker Compose + VPS | Предсказуемый запуск приложения и PostgreSQL |
| Мониторинг | Sentry + структурированные журналы | Сбор ошибок и проблемных действий |

## Модули версии 1.0

1. Управление коллективами и участниками.
2. База знаний по предметам.
3. Управление ресурсами.
4. Профиль пользователя.
5. Заявки на получение ресурсов.
6. Заявки на крафт предметов.
7. Калькулятор крафта.

Комнаты активностей и связанный с ними дашборд в версию 1.0 не входят.

## Что не стоит добавлять на старте

- микросервисы;
- Kubernetes;
- Redis без реальной потребности в очередях или кэше;
- отдельное мобильное приложение;
- сложный real-time слой.

Эти решения можно добавить позднее, если нагрузка или сценарии портала действительно этого потребуют.
