# Тестовый импорт предмета

Тестовый источник:

`https://corepunk.help/items/implant/implant-arcane-buster-t3?type=implant`

## Что импортируется

- основная запись предмета;
- все варианты качества и их изображения;
- слоты случайных вторичных характеристик: 0 для Common, 1 для Uncommon, 2 для Rare и 3 для Epic;
- характеристики и отдельные иконки характеристик;
- иконки, встроенные в описание эффекта;
- базовый рецепт и рецепты улучшения;
- полные записи всех прямых ингредиентов;
- профессия, уровень мастерства, цена, уровень, тир и слот.

Данные предметов пока сохраняются на английском языке. Русский перевод будет отдельным этапом и не изменит исходный снимок данных.

## Повторный запуск

```powershell
pnpm scrape:test-item
```

Результат сохраняется в `src/data/implant-arcane-buster-t3.json`, а изображения — в `public/game-assets`.

API Corepunk Help хранит варианты качества как базовую запись предмета и отдельные изображения. Импортёр нормализует их в самостоятельные записи, сохраняя ссылку на базовый предмет в поле `baseSlug`.

В карточке вариации качества кликабельны. Выбор грейда меняет основное изображение, активное качество и доступные secondary stats.
