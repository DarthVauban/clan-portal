# Локализация базы знаний

Английский импорт `src/data/corepunk-items.json` всегда остаётся неизменным источником данных.

Русская локализация хранится отдельно:

- `src/data/corepunk-items-ru.json` — переведённые строки;
- `src/localization/corepunk-glossary.json` — версионируемый словарь терминов;
- `scripts/translate-corepunk-items.mjs` — воспроизводимый перевод и валидация.

## Правила

- одинаковые игровые термины должны добавляться в словарь, а не исправляться отдельно в каждой карточке;
- slug, типы, связи рецептов и внутренние идентификаторы не переводятся;
- маркеры характеристик `[ap]`, `[armor]`, `[scale:...]` и HTML-разрывы строк защищаются от изменения;
- пользовательский переключатель RU/EN изменяет только названия предметов;
- выбранный язык названий сохраняется в `localStorage` браузера;
- описания и интерфейс всегда отображаются на русском языке.

## Обновление перевода

После обновления английской базы:

```powershell
pnpm translate:items
```

Скрипт использует локальный кеш и переводит только новые строки. Для принудительного полного обновления можно передать `--refresh`.

Перед записью итогового файла проверяется сохранность всех формул, маркеров и разрывов строк. Строки с повреждённой структурой блокируют публикацию локализации.
